# Loop

The `contidion` must be a boolean expression; assigments are not allowed.

## I. While Loop
```
while(condition){

}
```

#### Example
```
while(2 < 4){
  print("always run");
}
```

## II. For Loop

```
for(initialization; condition; updation){ 
}
```


#### Example
```
for(byte i = 0; i < 10; i++){
  print("always run");
}
```
